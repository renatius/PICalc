PI Calculator
=============

Poverty Index Calculator. 

Calculates the poverty index values as described in:

Mendola D., Busetta A., Milito A.M. (2011) Combining the intensity and sequencing of 
the poverty experience: a class of longitudinal poverty indices, 
Journal of the Royal Statistical Society – series A, 174(4), 953-973 
[http://onlinelibrary.wiley.com/doi/10.1111/j.1467-985X.2011.00699.x/abstract]
